Pair

Pair functions nearly identical to the pair provided by the standarad library, but there are slight differences. A JBON pair can hold any two values, and those types can be changed at runtime to whatever you want.

Constructors

Pair()

Default Constructor

Pair(std::string str)

This will parse the string and use it to initialize its 'key' and 'value'

Methods

template<typename T, typename U>
static Pair make(T key, U value);

This will make a pair with any two valid JBON values passed.

std::string serialize() const

This will return a serialized version of the instantiated pair.

void swap()

This will swap the 'key' and 'value' values.

template<typename T>
T& key();

This will get a refrence of the passed type from the 'key'.

template<typename T>
T& value();

This will get a refrence of the passed type from the 'value'.

template<typename T>
void setKey(const T& key);

This will change the 'key' to whatever key was passed.

template<typename T>
void setValue(const T& value);

This will change the 'value' to whatever value was passed.
