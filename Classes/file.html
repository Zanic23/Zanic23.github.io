File

File is the primary and suggested way to extract and parse jbon files.

Constructors

File(std::string filepath);

Will create a file object, and will parse through the text as the specified filepath.

File();

Default Constructor.

Methods

ClassCollection& getClasses();

This will return a refrence to the classes and objects parsed from the file in the form of a ClassCollection.

void include(std::string filepath);

Including a filepath is slightly different than loading a file. Including it will stictly parse the classes in that file. This is useful if you find yourself constantly rewriting the same same class template. Unlike load, which can only be used once per fileObject, you include as many files as you want.

void load(std::string filepath);

This will will load and parse the contents at the designated filepath.

void save();

Any changes done to the ClassCollection (that can be refrenced with ClassCollection) will be saved in the loaded filepath.
