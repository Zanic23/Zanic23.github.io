Array

A JBON array is just like an array from any other language. JBON arrays do differ, however, in fact that they can hold multiple different types at once.

Constructors

Array();

Default Constructor

Array(std::string str);

This will parse the string and construct the array appropriately.

Methods

template<typename T>
T& at(const int index);

This will return a refrence of the type passed, at the index passed.

template<typename T>
std::vector<T> getVector();

If all the values in the array of type T, this function will return a vector of type T.

template<typename T>
void push_back(const T& value);

Pass any valid JBON type, and it will add it to the JBON array.

template<typename...args>
static Array make(args...values);

Pass as many valid JBON type, and it will add this function will create an array with those values.

std::string serialize() const;

This will return a serialized version of the instantiated array.

iterator begin();

Will return a begin iterator for the array.

iterator end();

Will return an end iterator for the array.

const_iterator begin() const;

Will return a const begin iterator for the array.

const_iterator end() const;

Will return a const end iterator for the array.

void erase(iterator it);

Will delete the element from the array at the specified iterator.

int size();

Will return the size of the array.

bool empty();

If the array has no elements, this will return true, otherwise false.
