ObjectCollection

ObjectCollections are collections of Constructed JBON objects. ClassObjects are made strictly from class schemas, and so the only way to get an ObjectCollection is through a GenericClass.

Methods

ClassObject& operator[](std::string name);

This will return a refrence to an object with the specified name. If none can be found, this will throw an error.

iterator begin();

Will return a begin iterator for the collection.

iterator end();

Will return an end iterator for the collection.

const_iterator begin() const;

Will return a const begin iterator for the collection.

const_iterator end() const;

Will return a const end iterator for the collection.

iterator find(std::string name);

This will return an iterator to an object with the specified name. If none can be found, this will return an end iterator.

void erase(iterator it);

Will delete the element from the array at the specified iterator.

int size();

Will return the size of the collection.

bool empty();

If the collection has no elements, this will return true, otherwise false.
