SerializedClass<T>

A SerializedClass provides an interface between serialized C++ class instances parsed from a file. The passed type to the template, signifies what type of class instances it will hold.

Constructors

SerializedClass();

Default Constructor

SerializedClass(std::string schema);

This will parse the class schema and create the appropriate SerializedClass.

Methods

ObjectCollection& getObjects();

This will return a refrence to all constructed objects associated with this class. Any changes to the refrence will be remembered by the class, so remember to save the return value in a refrence. This is the only way for a user to get an ObjectCollection in current JBON.

T& operator[](std::string const& key);

This will return an instance of type T, with the specified name.

std::string name() const;

This will return the name of the class.

friend std::ostream& operator << (std::ostream& stream, SerializedClass& serializedClass);

This allows the SerializedClass to be passed to a standard stream.

std::vector<T>& getObjects();

returns a vector of instances of class T, populated from the JBON file. Any changes to these objects will be reflected if the class is saved.

void push_back(T const& value);

Will push back an Object of type T.

void makeAndAddObject(std::string_view parsed);

Pass a string view of the object you'd like parsed (without any spaces), and it will create and add an instance of T to its internal vector.

std::string serialize();

This will return the class, along with its objects transformed into text.

std::string serializeBody();

This will return only the classes's objects transformed into text.
