Value

Value is the base unit for JBON, and represents all possible valid JBON types.

Constructors

Value()

Default Constructor

Methods

static Value parse(std::string asString);

Will parse the string into a valid jbon type, and if it cannot will store an int initialized with 0.

std::string serialize() const

This will return the instantiated value as a string.

template<typename T> T& get();

This will return a refrence to the type specified by the template.

template<typename T> static Value make(T value);

As long as the type is a valid JBON type, the value will be created with the value.

template<typename T> bool holds();

If the Value is currently holding the type specified by the template, this will return true, otherwise false.

value_t& variant()

Willl return a refrence to the raw std::variant. The actual std::variant holds int, bool, float, std::string, and ValueType*.
