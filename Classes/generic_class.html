GenericClass

A GenericClass are the templates, or classes of JBON. They provide a schema which all of its objects must follow.

Constructors

GenericClass();

Default Constructor

GenericClass(std::string schema);

This will parse the class schema and create the appropriate GenericClass.

Methods

ObjectCollection& getObjects();

This will return a refrence to all constructed objects associated with this class. Any changes to the refrence will be remembered by the class, so remember to save the return value in a refrence. This is the only way for a user to get an ObjectCollection in current JBON.

std::string serialize() const;

This will return a serialized version of the GenericClass. This string will contain the class schema, as well as all of its serialized list of associated objects.

std::string name() const;

This will return the name of the class.

friend std::ostream& operator << (std::ostream& stream, GenericClass& genericClass);

This allows the GenericClass to be passed to a standard stream.

template<typename...args> ClassObject& constructObject(args...list);

Add an associated object to the respective class, initialized with the passed variable list. This will also return a refrence to the object added to the ObjectCollection.

template<typename...args> ClassObject& constructNamedObject(std::string name, args...list);

Add an associated object, with a name, to the respective class, initialized with the passed variable list. This will also return a refrence to the object added to the ObjectCollection.

ClassObject& constructObject(std::vector<Value> values)

Add an associated object to the respective class, initialized with the passed Values. This will also return a refrence to the object added to the ObjectCollection.

ClassObject& constructNamedObject(std::string name, std::vector<Value> values)

Add an associated object, with a name, to the respective class, initialized with the passed variables. This will also return a refrence to the object added to the ObjectCollection.

ClassObject& constructObject(std::string data);

Add an associated object to the respective class, initialized with the string that will be parsed. This will also return a refrence to the object added to the ObjectCollection.
