ClassCollection

Class container is a storage container for Generic Classes. It can be made manually, or recieved from a File object or parse function.

Constructors

ClassCollection();

Default Constructor.

Methods

viod push_back(const GenericClass& genericClass);

This method will push back a generic class into the class collection.

GenericClass& operator[](std::string name);

This will return a refrence to a generic class with the specified name.

int size();

Will return the number of GenericClasses contained in the ClassCollection.

std::string serialize() const;

This will return a serialized version of the ClassCollection. This serialized text includes the classes, as well as all of their objects.

friend std::ostream& operator << (std::ostream& stream, ClassCollection& collection);

This allows the ClassContainer to be passed into standard streams.
