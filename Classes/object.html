Object

A JBON Object is simply a set of key-value pairs, where the the key is always a string, and the value can be any valid JBON type.

Constructors

Object(;

Default Constructor

Object(std::string str);

This will parse the string and create the appropriate object from it.

Methods

std::string serialize(); const

This will return a serialized version of the object.

template<typename T> void emplace(std::string key, T value);

This will add a key value pair to the object, so long as T is a valid JBON type.

template<typename T> T& at(std::string key);

This will return a refrence to the type specified at the key passed.

template<typename T> void set(std::string key, T value);

This will change the value at the specified key to the new value passed.

void erase(std::string key);

This will delete the key value pair whose key matches with the one passed.

bool contains(std::string key);

Will return true if the object contains the key, false if not.

int size();

This returns the amount of key-value pairs held by the object.

friend std::ostream& operator<<(std::ostream& stream, const Object& object);

This allows Objects to be passed in streams such as std::cout.
