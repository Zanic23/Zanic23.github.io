Functions

jbon::ClassCollection parse(std::string text);

This is function will take any valid JBON text and parse through it, returning a collection of classes.

T parseString<T>(std::string str);

This template function will convert the passed string into the type passed. Any basic JBON type, including custom types, are valid.

void removeSpaces(std::string& str);

Will remove any whitespace from a string. This includes tabs, spaces, and newlines. This is primarily used when parsing, as whitespace does not affect JBON.

std::vector<std::string> split(std::string str, char delim = ',');

This function will split the passed string by the supplied delimeter (comma be default), and return the regexed string.

std::vector<std::string> split(std::string str, std::vector<std::pair<char, char>>, char delim = ',');

This function does the exact same as the other split function, however, it will not look for delimeters inside of any char pair passed. For example, the string "Hello, my favorite subjects are [Math, Science, History]" passed with pair {'[',']'} will return "Hello" and " my favorite subjects are [Math, Science, History]". You are also able to pass exclusion pairs that are the same like {'\"','\"'}.

std::vector<std::string> jbonSplit(const std::string& str, char delim = ',');

This function will split a string by the supplied delimeter, and it will ignore any JBON exclusion symbols. For example, it wouldnt do anything to the array "[2,3]", as the only comma is inside of a bracket.

void jbonFileSplit(std::vector<std::string>& objects, const std::string& str);

This function will any valid JBON text and split it into classes and class objects.

void indent(std::string& str, int spaces = 0);

indent will add a tab before at the begining of each new line.

bool is(Type)(std::string str);

There are four valid isType functions. They are, isInt(), isFloat(), isString(), and isBool(). If the passed string is the respective type, it will return true, otherwise false.
